# MELPKG
A simple commandline app for installing package on linux
# Installation
## Using Pip
```bash
  $ pip install melpkg
```
## Manual
```bash
  $ git clone https://github.com/MelaineGerard/melpkg
  $ cd melpkg
  $ python setup.py install
```
# Usage
```bash
$ melpkg
```
## Install
`install <package>`
```bash
$ melpkg install python
```
## Remove
`remove <package>`
```bash
$ melpkg remove python
```
## Search
`search <package>`
```bash
$ melpkg search python
```
## update
`update`
```bash
$ melpkg update
```
## upgrade
`upgrade`
```bash
$ melpkg upgrade
```
